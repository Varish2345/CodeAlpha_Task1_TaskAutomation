# Task Automation with Python

## Description

This project is developed as part of the CodeAlpha Python Programming Internship.

The program extracts all email addresses from a text file using Python Regular Expressions (Regex) and saves them into a new text file automatically.

## Features

* Reads data from a text file
* Detects valid email addresses
* Saves extracted emails into another file
* Uses Python file handling
* Uses Regular Expressions (re module)

## Technologies Used

* Python 3
* Regular Expressions (re)
* File Handling

## How to Run

1. Place `emails.txt` in the project folder.
2. Run:

   ```
   python task1.py
   ```
3. The extracted email addresses will be saved in `extracted_emails.txt`.
